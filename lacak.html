LACAK

Description

Aplikasi LACAK ini merupakan Aplikasi untuk memonitoring dan pendataan keuangan untuk perjalanan dinas di suatu Pemerintahan Kota. Aplikasi ini saya buat pada tahun 2022 pada saat saya bekerja di PT. Cosmo Mandiri Sejahtera. Dalam pembuatan aplikasi ini saya memakai Bahasa PHP dan menggunakan Framework CodeIgniter, dan dalam project ini saya membuat Aplikasi ini dari 0 / dari awal. Dan saya juga berperan sebagai FullStack Web Developer, merancang database nya di MySQL, serta men-Hosting nya di dalam Cpanel pada Aplikasi MOJA ini
